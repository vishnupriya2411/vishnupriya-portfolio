# P003 figures — miRNA-Gene Regulatory Networks in Schizophrenia

Reproducible generation of the two figures shown in the P003 project modal.
Rendered with **ggplot2 + svglite** (same toolchain as P001/P002) from data
transcribed verbatim from the project presentation's embedded result charts.

## Source
"Mapping of microRNA-mediated Gene Regulatory Networks across different tissues
in Schizophrenia" (Vishnu Priya Nuthanapati, CAMPUS Fellowship 2020; poster
presented at Experimental Biology 2021). The `.pptx` lives in the
[Transcriptomics](https://github.com/vishnupriya2411/Transcriptomics) repo.

Values in `data/` were read directly from the charts embedded in the deck:
- chart 4 + slide 21 → `gene_drug_targets_ba46.csv` (hub gene log2FC + matched drug)

## Figures shown in the P003 modal
| Figure | Type | Source |
|---|---|---|
| `p3_network_ba46.png`  | Real Cytoscape network (BA46 down-regulated miRNAs → target genes) | slide 13 of the deck, copied verbatim |
| `p3_drug_targets.svg`  | ggplot bar: 7 dysregulated hub genes → 4 repurposable drugs | generated by this script |

Only `p3_drug_targets.svg` is generated here; the network image is a real figure
from the presentation and is not re-rendered.

## Reproduce
```bash
cd figures/p003
Rscript make_p3_figures.R
```
Requires R with `ggplot2` and `svglite`.

## Key numbers from the deck (used in the portfolio)
- 5 tissues profiled: prefrontal cortex BA46 & BA10, superior temporal cortex BA22, PBMCs, iPSCs
- BA46 network: 2,813 genes, 50 miRNAs (Cytoscape)
- top hub miRNA: hsa-miR-590-5p (targets 587 genes)
- 4 candidate repurposable drugs across 7 dysregulated hub genes
